# 배송 담당자 문의사항

## 1. 포장 및 택배사 기준

### 퍼즐매트 포장 방법
- **현재 알고 있는 내용**: 퍼즐매트는 "Only 비닐" 포장으로 알고 있습니다.
- **질문**: 
  - 비닐 포장 시 **최대 몇 장(또는 몇 세트)**까지 한 번에 포장이 가능한가요? 
  - 특정 수량을 넘어가면 박스 포장으로 변경되나요, 아니면 비닐을 여러 개로 나누나요?

### 택배사 구분 (로젠 vs 경동)
- **현재 적용된 로직**: 
  - 대형 화물(대박스, 비닐 포장 포함) → **경동택배**
  - 소형 화물(소박스 only) → **로젠택배**
- **질문**: 
  - 이 구분 기준이 현재도 유효한가요?
  - 예외 사항(예: 무게 기준, 특정 지역 등)이 있다면 무엇인가요?

---

## 2. 배송비 관련

### 제품별 배송비 책정
- **질문**: 각 제품(롤매트 사이즈별, 퍼즐매트 등)의 **최근 배송비 단가표**를 받을 수 있을까요?
  - (프로그램 내에 배송비 계산 또는 예측 로직을 넣기 위함인지, 단순 참고용인지 확인 필요)

### 배송비 정리 업무 필요 여부
- **상황**: 스마트스토어 주문 파일에는 '고객이 결제한 배송비(배송비 합계)'가 포함되어 있습니다. (대부분 무료배송(0원)이거나 묶음 배송비 5,000원 등)
- **질문**: 발주 리스트를 넘겨드릴 때, **우리가 실제로 부담해야 할 택배비(예상치)**를 별도로 계산해서 정리해드려야 하나요?
  - 아니면 주문 파일에 있는 '결제된 배송비' 정보만 그대로 유지하면 되나요?
  *(참고: 현재 파일상에는 고객 결제 금액만 나와 있어, 실제 택배사 정산 금액과는 차이가 있을 수 있습니다.)*

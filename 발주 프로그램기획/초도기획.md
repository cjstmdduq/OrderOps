# 발주 프로그램 초도기획

## 목적
발주작업을 간편화 하는 것

## 사용방식
사용자는 별도 환경설정 없이 웹에서 엑셀을 올린다!
변환되서 확인할 수 있고 다운로드도 손쉽게!

## 기술스펙
1. 프론트엔드 - HTML/CSS/JS 순수 (또는 React)
2. 백엔드 - Python (엑셀 파싱)
3. 배포 - 정적 웹호스팅 가능

## 로직 구상

### 1. 엑셀 파싱
사용자가 업로드한 엑셀파일(xlsx, csv)을 파싱
- 입력: 스마트스토어 원본 주문서
- 처리: 상품DB로 VLOOKUP (디자인, 길이 추출)

### 2. 송장화
택배사 양식에 맞게 변환
- 경동택배: 송장화1 → 경동발주서
- 로젠택배: 송장화2 → 로젠양식

### 3. 사은품 대상자 구별
결제금액에 따라 구분
- 20만원+: 실리콘 테이프
- 30만원+: 발매트 추가

### 4. 패킹알고리즘 (핵심)
- **비닐포장**: 140폭 8m 이상 → ★마킹
- **박스포장**: 중소형 롤매트 → 소박스/대박스
- **합포장**: 소형 상품/테이프 → 합 표시
- **N-n 라벨링**: 같은 사람에게 여러 박스 시

### 5. 재단요청 감지
배송메세지에서 키워드 검출
- "재단", "커팅", "절단", "cm" 등

## 입력 파일
```
스마트스토어_전체주문발주발송관리_YYYYMMDD_HHMM.csv
```

## 출력 파일
- 경동발주서 (xlsx)
- 로젠양식 (xlsx)
